# MSVR MC 文档查看器使用说明

## 📋 概述

新的文档查看器支持直接读取和渲染 Markdown 文件，提供了更好的阅读体验和更多实用功能。

## 🚀 主要功能

### 1. Markdown 渲染
- ✅ 支持标准 Markdown 语法
- ✅ 代码语法高亮显示
- ✅ 表格渲染
- ✅ 标题层级结构
- ✅ 列表和引用块

### 2. 阅读功能
- 📑 自动生成文档目录
- 🔍 快速导航到指定章节
- 🔤 字体大小调节（A+ / A-）
- 🖨️ 打印功能
- 📱 响应式设计，支持移动端

### 3. 快捷操作
- ⌨️ 键盘快捷键支持
- 🔄 一键刷新文档
- 📋 文档切换

## 📖 使用方法

### 1. 访问查看器
在浏览器中打开：`Viewer.html`

### 2. 选择文档
在左侧文档列表中点击想要查看的文档名称

### 3. 使用功能
- **目录导航**：点击"📑 目录"按钮显示/隐藏文档目录
- **字体调节**：使用 A+ / A- 按钮调节字体大小
- **打印文档**：点击"🖨️ 打印"按钮打印当前文档
- **刷新文档**：点击"🔄 刷新"按钮重新加载当前文档

### 4. 键盘快捷键
- `Ctrl + R`：刷新当前文档
- `Ctrl + P`：打印当前文档
- `Ctrl + +`：增大字体
- `Ctrl + -`：减小字体

## 📁 可用文档

当前支持以下 Markdown 文档：

1. **宪法.md** - MSVR MC 服务器宪法
2. **刑法.md** - MSVR MC 服务器刑法  
3. **民法典.md** - MSVR MC 服务器民法典
4. **立法法.md** - MSVR MC 服务器立法法
5. **经济指南.md** - MSVR MC 服务器经济指南
6. **README.md** - 使用说明文档

## 🔧 技术说明

### 使用的第三方库
- **marked.js** - Markdown 解析器
- **Prism.js** - 代码语法高亮
- **原生 JavaScript** - 功能实现

### 浏览器兼容性
- ✅ Chrome (推荐)
- ✅ Firefox
- ✅ Safari
- ✅ Edge
- ✅ 其他现代浏览器

## ⚠️ 注意事项

1. **文件路径**：确保 Markdown 文件位于 `md/` 目录下
2. **网络连接**：需要网络连接以加载第三方库
3. **文件编码**：Markdown 文件应使用 UTF-8 编码
4. **文件权限**：确保浏览器有权限读取本地文件

## 🛠️ 故障排除

### 文档加载失败
- 检查文件路径是否正确
- 确认文件是否存在
- 检查网络连接是否正常
- 查看浏览器控制台错误信息

### 格式显示异常
- 刷新页面重试
- 检查 Markdown 语法是否正确
- 确认使用了标准 Markdown 格式

### 功能无法使用
- 确认浏览器支持 JavaScript
- 检查是否有浏览器插件冲突
- 尝试使用不同的浏览器
