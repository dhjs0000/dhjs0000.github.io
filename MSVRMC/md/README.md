## 使用说明

### 主要用途
- **便于阅读**：Markdown 格式在各种设备上都有良好的阅读体验
- **便于编辑**：使用任何文本编辑器都可以轻松修改
- **版本控制**：便于跟踪文档的修改历史
- **格式转换**：可以轻松转换为 HTML、PDF 等其他格式

### 编辑建议
- 使用支持 Markdown 的编辑器（如 VS Code、Typora 等）
- 保持原有的法律条文结构和编号
- 修改后请及时更新文档的修订日期
- 重大修改建议先在副本上进行测试

### 格式规范
- 标题使用 `#` 符号，层级清晰
- 法律条文使用粗体标注，如：**第一条**
- 列表使用标准的 Markdown 列表语法
- 表格用于展示结构化信息
- 引用和说明使用引用块语法
